Change default connection alias
The default messaging connection alias determines the messaging provider to which messages are sent when the Connection alias name property for the publishable document type is set to DEFAULT or is blank.
Click Return to webMethods Messaging Settings to return to the previous page.
Select New Default Connection Alias
Connection Alias Name
Select the name of the messaging connection alias that you want to use as the new default messaging connection alias.
Update
Saves the selected alias as the new default messaging connection alias.
