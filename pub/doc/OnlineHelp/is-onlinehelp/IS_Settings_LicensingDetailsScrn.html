Sales Information
Includes the product license key, which is a special code associated with your license. This key is stored in the Integration Server_directory \instances\instance_name\config\licenseKey.xml file.
The customer and partner fields typically contain your company name and your partner's company name.
Product Information
Includes the license expiration date and the operating systems this Integration Server is licensed to run on.
Integration Server
Lists the Software AG code for this product, the product version number, a list of features (with check marks next to the ones you are licensed to use), and the maximum number of concurrent sessions allowed to run on your server.
License Information
Reflects the number of CPUs (Price Quantity) and the number of core processors per CPU (Price Unit) that this Integration Server is licensed to run on.
Trading Networks
If Integration Server is configured to run with Trading Networks, shows the maximum number of partners Trading Networks allows.
