webMethods Messaging Connection Alias Definitions
Displays the messaging connection aliases that are configured for the Integration Server.
Default
Indicates which messaging connection alias is the default messaging connection alias for the Integration Server. The default messaging connection alias determines the messaging provider to which messages are sent when the Connection alias name property for the publishable document type is set to DEFAULT or is blank.
Type
Indicates the webMethods messaging provider used with the messaging connection alias. The Type can be Broker (Deprecated), Universal Messaging, or Digital Event Services.
Connection Alias Name
Name of the messaging connection alias.
Description
A description of the messaging connection alias.
CSQ Count
Specifies the number of documents in the client side queue for the messaging connection alias. For a Broker connection alias, the client side queue is also called the outbound document store.
Note:
The CSQ Count field does not apply to the IS_DES_CONNECTION alias as Integration Server does not maintain a client side queue for the alias. Instead, Digital Event Services maintains a “Store and Forward Queue”. To view the number of outbound messages in the Store and Forward Queue, also referred to as the progress queue (IPQ), associated with the alias, use Command Central.
Enabled
Displays the status of the messaging connection alias. When a messaging connection alias is enabled, Integration Server can use it to send and receive messages from the messaging provider on behalf of publishing services and webMethods messaging triggers, respectively. A disabled messaging connection alias cannot be used to send messages to or receive messages from the messaging provider specified in the alias. To disable a messaging connection alias, click the Yes link in this column. To enable a messaging connection alias, click the No link in this column.
Delete
Deletes the messaging connection alias specified in Connection Alias Name. To delete a messaging connection alias, click the icon in the Delete column for the alias.
