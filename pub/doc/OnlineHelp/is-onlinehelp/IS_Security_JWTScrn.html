The JWT page displays the settings that Integration Server requires to use the JSON Web Token (JWT) authentication for service requests.
Click Trusted Issuers to view and add the trusted issuers.
Click Issuer Configuration to manage the issuer-certificate mapping and issuer-skew mapping.
Click Edit Global Claim Settings to add or edit the global settings.
Audience
Specifies the audience value to use for this Integration Server. Integration Server validates this value against the Audience (aud) claim in the incoming JWT. Provide a valid string or URI as the audience value.
Max Global Skew (in seconds)
Specifies the maximum number of seconds for which the JWT issuer and Integration Server clocks can differ and still allow JWT expiry (exp) and not before (nbf) validations to succeed.
