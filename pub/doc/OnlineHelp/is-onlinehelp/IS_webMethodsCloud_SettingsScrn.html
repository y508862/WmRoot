Tenant connections
 
Tenant alias
The Tenant connections page displays the tenant aliases that you use to specify the location, login, and certificate details for 2-way SSL configuration of Integration Cloud for which you will define accounts and applications.
Note:
See the Configuring On-Premise Integration Servers for webMethods Cloud document for more information.
To open the Tenant connections screen, click Tenant connections under the webMethods Cloud menu on the Navigation pane.
Tenant connections
Alias
Name for the tenant.
User Name
The user name the on-premise Integration Server must use to access Integration Cloud.
Tenant URL
The URL of Integration Cloud with which to share accounts and applications created on the on-premise Integration Server
Test
Click to test the tenant connection to Integration Cloud.
Enabled
Whether the tenant is enabled or disabled.
Delete
Click to delete the tenant.
