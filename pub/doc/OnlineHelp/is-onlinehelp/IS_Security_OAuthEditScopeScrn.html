Edit Scope
Use the Edit Scope page to modify existing scopes.
Click Return to Scope Management to return to the Scope Management.
Scope Configuration
Name
Specifies a unique name for the scope. You cannot change the Name field for a scope.
Description
A description of the scope.
Folders and services
Specifies the list of folders and individual services that the client can access on behalf of the resource owner. You can add more than one folder or service at a time by specifying multiple lines, one folder or service to a line. Press ENTER to separate lines.
Note:Integration Server validates the entries. If you enter a folder or service that does not exist, Integration Server does not add the scope.
URL templates
Specifies a URL template for each HTTP method. The URL template for each HTTP method must be mapped to each scope. The format of URL template is <HTTP method>_<rad namespace>/<URL template of resource>. For example, GET_RadNamesSpace/employee.
Save Changes
Updates Integration Server with the new scope settings. Your changes take effect immediately.
