Edit server alias
Use the Edit server alias page to edit an alias that has been defined for an SFTP server.
You can edit only the Host Name or IP Address, Port Number, and Host Key Location properties. You cannot leave the Host Name or IP Address and Port Number fields empty.
Click Return to SFTP to return to the list of SFTP server aliases defined in your Integration Server.
SFTP Server Alias Properties
Host Name or IP Address
Update the host name or IP address of the SFTP server. An SFTP server alias name:
Can contain only underscores (_) and periods (.) as special characters.
Cannot begin with the string "http://".
Can be of a maximum length of 255 characters.
Port Number
Update the port number of the SFTP server. The port number must be within the range of 0 and 65535 (inclusive).
Proxy Alias
Specify the proxy alias through which the request is to be routed. The proxy alias can be HTTP, HTTPS, or SOCKS. If a proxy alias is not specified, Integration Server makes outbound requests using each enabled proxy server alias until the request is sent successfully or all proxy servers have been tried.
Host Key Fingerprint
Displays the public key fingerprint of the SFTP server that was previously defined.
Host Key Location
In case you have modified the SFTP server details, update the location of the public key of the updated SFTP server.
Important:
The public key file must be present on the same machine in which you have installed Integration Server.
Get Host Key
If you do not have the public key of the SFTP server, click Get Host Key. Integration Server will retrieve the public Key for the host and port you specified and save it in a temporary folder. Integration Server will then display the path to the retrieved public key file in the Host Key Location field.
Min DH Key Size
The minimum DH key size. This parameter is not applicable to SFTP Client Version 1.
Note:
If you see the error message: "com.maverick.ssh.SshException: Minimum DH p value not provided [2048]", set Min DH Key Size to 2048.
Max DH Key Size
The maximum DH key size. This parameter is not applicable to SFTP Client Version 1.
Note:
Key Exchange Algorithms such as diffie-hellman-group-exchange-sha1 and diffie-hellman-group-exchange-sha256 consider values set for the Min DH Key Size and Max DH Key Size parameters. All other algorithms ignore these values.
Preferred Key Exchange Algorithms
Update your choice of algorithms that Integration Server presents to the SFTP server for key exchange. Specify the order in which Integration Server presents the algorithms to the SFTP server by clicking Up or Down. The SFTP server has its own set of preferred algorithms configured. At the time of key exchange, one of the algorithms supported by both Integration Server and the SFTP server is chosen.
Preferred MAC Algorithms S2C
Update your choice of server-to-client Message Authentication Code (MAC) algorithms. If you want to reorder an algorithm in the list, select the algorithm, and click Up or Down. If you do not want to use an algorithm in the list, select the algorithm, and click Right. The algorithm moves to the list of excluded algorithms.
Preferred MAC Algorithms C2S
Update your choice of client-to-server Message Authentication Code (MAC) algorithms. If you want to reorder an algorithm in the list, select the algorithm, and click Up or Down accordingly. If you do not want to use an algorithm in the list, select the algorithm, and click Right. The algorithm moves to the list of excluded algorithms.
Preferred Ciphers S2C
Update your choice of preferred server-to-client ciphers. If you want to reorder an algorithm in the list, select the algorithm, and click Up or Down accordingly. If you do not want to use an algorithm in the list, select the algorithm, and click Right. The cipher moves to the list of excluded ciphers.
Preferred Ciphers C2S
Update your choice of preferred client-to-server ciphers. If you want to reorder an algorithm in the list, select the algorithm, and click Up or Down accordingly. If you do not want to use an algorithm in the list, select the algorithm, and click Right. The cipher moves to the list of excluded ciphers.
Save Changes
Updates the SFTP server alias definition.
