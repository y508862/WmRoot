Edit Mime types
Use the Edit mime types page to change the content handler mappings in the mime.types configuration file.
Click Return to Mime Types Settings to return to the list of current content-type settings.
Click Save Changes to save any changes to the mime type mappings in the mime.types configuration file. After you have saved the changes, return to the Mime types and click Reload Mime Type Settings to reload the mime.types file without restarting Integration Server.
Content-Type File Extension(s)
Edit the mime.types configuration file in the Integration Server_directory \instances\instance_name\lib directory. These settings map the HTTP Content-Type files to the listed extensions.
