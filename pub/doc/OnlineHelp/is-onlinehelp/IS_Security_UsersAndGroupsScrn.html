Use the User management page to change the membership of a user group.
Note:
You may assign a user to a group in one of two ways: you can select a user name first and add that user to any of the available groups, or you can select a group name first and selects its members from a list of user names. Both procedures produce the same result. The use of one over another is strictly a matter of personal preference or convenience.
Central User Management Configuration
The Central User Management field displays whether or not the server is configured to access user account and group information from an external directory. For information about setting up central user management, see webMethods Integration Server Administrator’s Guide.
Local User management
Use these fields to manage user account and group information.
Click Add and Remove Users to add a new user name to the server or delete one that already exists.
Click Add and Remove Open ID Provider Users to add, edit, or remove an OpenID Provider user.
Click Add and Remove Groups to add a new user group to the server or delete one that already exists.
Click Enable and Disable Users to temporarily block a user from the server and to re-enable a user whose access has been blocked.
Click Password Restrictions to specify the criteria for passwords that are used on your server.
Click Password Expiration Settings to specify the password expiration settings for non-administrator users.
Click Account Locking Settings to configure the account locking settings for non-administrator users.
Click LDAP Configuration to configure the server to access user account and group information from an external directory.
Users
Use this section to select a user name and then specify the groups to which it belongs.
Select User
Lists the user names that are defined on your server. To display the list of groups to which a user belongs, select the name of the user from this list.
(change password)
Opens the Edit User Information page, which allows you to reset the selected user's password.
Note:
Select Allow Digest Authentication to use password digest as an authentication option.
Groups user belongs to
Specifies the groups to which the selected user belongs. Use the -> and <- buttons to move group names on or off this list.
Remaining Groups
Specifies the groups to which the selected user does not belong. Use the -> and <- buttons to move group names on or off this list.
Use this section to select a group and then specify the users that belong to the group.
Select Group
Lists the group names that are defined on your server. To view or edit the membership of a group, select the group from this list.
Users in this Group
Specifies the list of users that belong to the selected group. Use the -> and <- buttons to move user names on or off this list.
Remaining Users
Specifies the list of users that do not belong to the selected group. Use the -> and <- buttons to move user names on or off this list.
Save Changes
Applies the changes to the server. The changes take effect immediately.
