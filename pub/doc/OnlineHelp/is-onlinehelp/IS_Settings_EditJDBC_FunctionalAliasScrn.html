Edit JDBC functional alias assignment
Use the Edit JDBC functional alias assignment page to associate a functional alias with a connection pool.
Click Return to JDBC Pool Definitions to return to the previous page.
Function Name
Functional alias to associate with a connection pool. You cannot edit this field.
Function Description
Brief description of the functional alias. You cannot edit this field.
Associated Pool Alias
Connection pool you want the functional alias to use.
Fail-Fast Mode Enabled?
Indicates whether or not the JDBC functional alias is configured to enter fail-fast mode when a transient error caused by an unavailable database prevents the connection pool alias used by the JDBC functional alias from establishing a connection to the database. Select Yes to enable fail-fast mode. Select No to disable fail-fast mode.
Note:
Fail-fast mode cannot be enabled for the CentralUsers function. Consequently, the Fail-Fast Mode Enabled? field does not appear when editing the CentralUsers functional definition.
Currently in Fail-Fast?
Indicates whether or not the JDBC functional alias is currently in fail-fast mode. Yes indicates the alias is in fail-fast mode. No indicates the alias is not in fail-fast mode.
Note:
Fail-fast mode cannot be enabled for the CentralUsers function. Consequently, the Currently in Fail-Fast? field does not appear when editing the CentralUsers functional definition.
Save Settings
Applies changes made to the Associated Pool Alias setting.
