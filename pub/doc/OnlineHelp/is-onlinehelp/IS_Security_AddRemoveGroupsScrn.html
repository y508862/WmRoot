Add and Remove Groups
Use the Add and Remove Groups page to add a new group to the server or delete an existing one.
Click Return to User Management to return to the list of groups and users that are currently defined on your server.
Create Groups
Use this section to create a new, empty group.
Group Names
Specifies the name of the group that you want to add. If you want to add more than one new group, type each name on a separate line.
Create Groups
Saves the group names specified in Group Names.
Remove Groups
Use this section to delete a group from your server.
Select Groups
Lists the group names that are defined on your server. To delete a group, click the check box next to its name. Note that you cannot delete the system group names.
Remove Groups
Deletes the group names that you have selected in the Select Groups list.
