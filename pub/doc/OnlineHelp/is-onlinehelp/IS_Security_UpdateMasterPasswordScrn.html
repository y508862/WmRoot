Update Master Password
Use the Update Master Password page to change the master password for outbound passwords.
Note:
To keep outbound passwords synchronized with the master password, the Integration Server does not process requests to store and retrieve outbound passwords when the master password is being changed. Therefore, if your system has many aliases, consider performing the master password change during off peak hours to prevent any decrease in performance.
Click Reset all Outbound Passwords to reset the master password to its default setting and to clear all outbound passwords.
Important:
Reset the outbound passwords only if your Integration Server encrypts outbound passwords using PBE and you have lost your master password. Once you reset the passwords, the Integration Server will no longer be able to connect to the applications and subsystems for which the passwords are required, until you manually reset them.
Click Return to Outbound Passwords to return to the Outbound Passwords page.
Current Password
Enter the current master password.
New Password
Specifies the new password. Re-enter it to confirm it is correct. Click Change Password to apply the new password.
