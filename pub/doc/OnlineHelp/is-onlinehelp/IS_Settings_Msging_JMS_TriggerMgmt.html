Use the JMS triggers page to view global and individual controls for JMS triggers.
Click Edit JMS Global Trigger Controls to change the number of server threads used for JMS triggers.
Click Refresh Page to update the information on the page.
Click Search Triggers to search for a trigger using the trigger name, JMS connection alias, or destination as search criteria.
Global JMS Trigger Controls
Use this section to view the view the maximum number of threads that can be used for JMS triggers and to view the adjusted concurrent processing level for concurrent triggers.
Thread Pool Throttle
Displays the percentage of the server thread pool that can be used for JMS triggers.
Processing Throttle
Displays the percentage of maximum allowed execution threads for concurrent JMS triggers.
Individual Standard JMS Trigger Controls
Use this section to view detailed information for each standard JMS trigger on Integration Server.
Click Go to Individual SOAP JMS Trigger Controls to jump to the Individual SOAP JMS Trigger Controls table.
Note:Integration Server Administrator displays the Go to Individual SOAP JMS Trigger Controls link only if the Integration Server has a loaded package that contains a SOAP-JMS trigger.
Trigger Name
Fully qualified name of the JMS trigger.
If appears next to a trigger name, the trigger is the parent of a trigger group. This row functions as a summary for the triggers in the trigger group. Click to expand and view all the triggers in the trigger group.
State
Indicates whether the trigger is enabled, disabled, or suspended.
If this is the trigger group header, the State column displays a summary of the state of all the triggers in the trigger group.
If all triggers in the trigger group have the same state, then the trigger group header will display that state.
If at least one trigger in the trigger group is enabled, then the trigger group header will display "Enabled [x of n]" where x is the number of enabled triggers and n is the total number of triggers in the trigger group.
If no triggers are enabled but at least one trigger in the group is Suspended, then the trigger group header will display "Suspended [x of n]" where x is the number of suspended triggers and n is the total number of triggers in the trigger group.
edit all
Click to access the Edit all JMS triggers in which you can enable, disable, or suspend all the JMS triggers on the Integration Server.
Status
The status of the JMS trigger, which can be “Running” or “Not Running (reason)” where reason identifies why the rigger is not running, such as “Not Running (trigger disabled)”.
If the listed JMS trigger is the trigger group header, the Status column displays a summary of the status of all the triggers in the trigger group.
If all of the triggers in the group are running, the Status column displays “Running “.
If at least one trigger in the group is running, the Status column displays “Running [x of n]”.
If none of the triggers in the group are running, the Status column displays “Not Running”.
Connection Alias Name
Name of the JMS connection alias that Integration Server uses to retrieve messages for the JMS trigger.
If this is the parent of a trigger group, the Connection Alias Name column displays a comma-separated list of the JMS connection aliases used by the triggers in the trigger group.
Destination(s)
The queues or topics on the JMS provider for which the JMS trigger receives messages.
If this is the parent of a trigger group, the Destination(s) column displays a comma-separated list of the destinations from which triggers in the trigger group receive messages.
Processing Mode
Message processing mode for the JMS trigger.
Processing Mode
Description
Serial
Integration Server processes messages received by the trigger one after the other.
Concurrent
Integration Server processes multiple messages for this trigger at one time.
Maximum Threads
Indicates the maximum number of server threads that can process messages for the JMS trigger concurrently. If the current maximum threads value is different from the configured maximum threads value (that is, the Execution Threads Throttle is set to less than 100%), the configured maximum threads value is displayed in parenthesis.
Current Threads
Specifies the current number of server threads processing messages for this trigger.
Individual SOAP JMS Trigger Controls
Use this section to view detailed information for each SOAP-JMS trigger, including WS (web service) endpoint triggers on Integration Server.
Note:Integration Server Administrator displays the Individual Standard JMS Trigger Controls table only if the Integration Server has a loaded package that contains a SOAP-JMS trigger.
Click Go to Individual Standard JMS Trigger Controls to jump to the Individual Standard JMS Trigger Controls table.
Trigger Name
Fully qualified name of the SOAP-JMS trigger. Note that WS endpoint triggers use the format WS Endpoint Trigger: alias where alias is the name of the provider web service endpoint alias that uses the WS endpoint trigger.
State
Indicates whether the trigger is enabled, disabled, or suspended.
