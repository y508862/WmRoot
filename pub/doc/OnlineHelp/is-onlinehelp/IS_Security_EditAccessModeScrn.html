Edit Access mode
Use the Edit Access mode page to view or modify the access mode settings for a port. The access mode settings determine which services are allowed to run on a port.
Click Return to Ports to return to the list of ports that are defined on the server.
Click Add Folders and Services to Deny List to specify services that are prohibited from running on this port. This link is shown only when the access mode is set to Allow by Default.
Click Add Folders and Services to Allow List to specify services that are permitted to run on this port. This link is shown only when the access mode is set to Deny by Default.
Click Set Access Mode to Allow by Default to permit all services except those you explicitly prohibit. If the access mode is currently Deny by Default, the services in the allowed list are cleared when you switch modes.
Click Set Access Mode to Deny by Default to prohibit all services except those you explicitly allow. If the access mode is currently Allow by Default, the services in the denied list are cleared.
Click Reset to default access settings to return the access mode settings to their default values.
Port Service Access Settings
Displays the current access mode. To switch modes, click the Set Access Mode to Deny by Default or Set Access Mode to Allow by Default link at the top of the screen.
Allow List
Shown only when access mode is set to Deny by Default. Displays the list of services that are permitted to be run from this port. The server will reject all requests for services that are not on this list. To add an entry to the list, click the Add Folders and Services to Allow List link. To remove an entry from the list, click the icon in the Delete column.
Deny List
Shown only when access mode is set to Allow by Default. Displays the list of services that are not allowed to be run from this port. The server will accept only requests for services that are not on this list. To add an entry to the list, click the Add Folders and Services to Deny List link. To remove an entry from the list, click the icon in the Delete column.
