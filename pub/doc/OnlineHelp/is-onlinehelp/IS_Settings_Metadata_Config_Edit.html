Edit Metadata Configuration
Screen Heading: Asset Publisher > Configuration > Edit
Use the Edit Asset Publisher Configuration screen to change your CentraSite connection settings.
To open the Edit Asset Publisher Configuration screen, go to the Metadata Configuration Screen and click Edit Configuration.
Click Return to Configuration to return to the previous page.
Central Configurator
IS Identifier
The name or the IP address of the machine on which this Integration Server is running. By default, Integration Server populates this field with the IP address of the machine. You can, however, enter a name here, which Integration Server will treat as an alias. In addition, Integration Server includes this name with the published metadata. You can use any name, but it cannot include the following illegal characters:
#-&@^!%*:$./\\`;,~+=)(|}{][><".
CentraSite URL
The url of the CentraSite system to which to publish metadata about Integration Server assets.
User Name
The name of the user account on CentraSite that will be used for publishing and retracting metadata.
Password
Password for User Name.
Save Changes
Saves the configuration settings. It is not necessary to restart Integration Server.
