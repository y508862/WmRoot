Health indicator properties
Use the HealthIndicator properties page to view the properties for the health indicators installed with Microservices Runtime. Only some of the health indicators have configurable properties.
Click Return to Indicators to return the list of health indicators.
HealthIndicator Health Indicator Property
Displays the configurable properties for the health indicator.
Health Indicator
Property Name
Value
Cluster
Number of cluster hosts
Minimum number of cluster members that must be available for the Cluster health indicator to return a status of UP. When the number of servers in the cluster is less than the specified minimum number, the Cluster health indicator returns a status of DOWN. The default is 2.
Diskspace
Free disk space threshold (as percentage of maximum available disk space)
Percentage of available free disk space on the Microservices Runtime host machine or container at or below which the Diskspace health indicator returns a DOWN status.
Memory
Free memory threshold (as percentage of maximum memory)
Percentage of available JVM memory for Microservices Runtime at or below which the Memory health indicator returns a DOWN status.
ServiceThread
Available threads threshold (as percentage of maximum server threads)
Percentage of available threads in the server thread pool at or below which the ServiceThread health indicator returns a DOWN status.
Sessions
Used licenses threshold (as percentage of total licensed sessions)
Percentage of used licenses at or below which the Sessions health indicator returns a DOWN status.
Edit
Click Edit to edit the HealthIndicator property value.
