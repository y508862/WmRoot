Edit gRPC channel
Use the Edit gRPC channelName page to edit the properties of a gRPC channel.
Click Return to gRPC Settings to return to the previous page.
gRPC Channel Properties
Name
Name of the gRPC channel. This property is read-only for Integration Server version 10.11.
Description
Text describing the gRPC channel. This property is read-only for Integration Server version 10.11.
Port
The HTTP/2 port the channel uses to receive and send messages. The default is 50051. If you change the port number, Integration Server unregisters all of the gRPC descriptors and their gRPC services from the old port and registers them with the new port.
Package
Package in which the gRPC channel is saved. This property is read-only for Integration Server version 10.11.
Default
Whether this channel is the default gRPC channel. For Integration Server version 10.11, only one gRPC channel is allowed and it must be the pre-defined gRPC channel GRPCDefault. This property is read-only for Integration Server version 10.11.
Enabled
Whether the gRPC channel is enabled. Select Yes to enable the channel. Select No to disable the channel.
gRPC Channel Credentials
Keystore Alias
The alias for the keystore that contains the key alias used by the gRPC server for SSL/TLS support.
Key Alias
The alias to the key entry that contains the private key used by the gRPC server for its SSL/TLS support. This key alias contains the private key used by the server as well as the certificate chain used to sign that certificate. The key alias must be in the keystore specified in Keystore Alias.
Truststore Alias
The alias for the truststore that contains the certificate authority (CA) certificates.
Save Changes
Click to save changes to the gRPC channel.
