Shut down or restart
Use the Shut down or restart screen to schedule a complete shutdown of the server, to shut down and restart the server in active mode, or to shut down and restart the server in quiesce mode. You can perform these options immediately or specify a time period before shutdown or restart.
Click Current Sessions to view the active sessions in webMethods Integration Server.
Shut down or restart...
Option
Description
After all client sessions end
Wait a specified number of minutes, or until all current non-Administrative sessions are complete (whichever comes first).
Immediately
Perform the requested action (shutdown or restart) immediately.
Shut Down
Shuts down webMethods Integration Server in the manner specified in Shut Down or Restart.
Restart
Shuts down webMethods Integration Server in the manner specified in Shut Down or Restart and starts it again in active mode.
Restart in Quiesce Mode
Shuts down webMethods Integration Server in the manner specified in Shut Down or Restart and starts it again in quiesce mode.
